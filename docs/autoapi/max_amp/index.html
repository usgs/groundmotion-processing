Max_Amp

Class for the maximum amplitude clipping detection algorithm.
class max_amp.Max_Amp(st, max_amp_thresh=6000000.0, test_all=False)#

Bases: gmprocess.waveform_processing.clipping.clip_detection.ClipDetection

Class for the maximum amplitude clipping detection algorithm.

st#

Record of three orthogonal traces.

Type:

StationStream

test_all#

If true, compute and store max amp for all traces.

Type:

bool, default=False

is_clipped#

True if the record is clipped.

Type:

bool

max_amp_thresh#

A threshold for the absolute maximum amplitude of the trace.

Type:

float, default=6e6

max_amp#

The maximum amplitude of the first clipped trace or list of max amps for each trace (if test_all=True).

Type:

float/list

See parent class.
_clean_trace(tr)#

Helper function to clean a trace (no normalizing in this algorithm).

Parameters:

tr (StationTrace) A single trace in the record.

Returns:

Cleaned trace.

Return type:

clean_tr (StationTrace)

_detect(tr)#

If any point exceeds the threshold, fail the trace. Threshold is given as max_amp_thresh, the absolute maximum amplitude of the trace.

Parameters:

tr (StationTrace) A single trace in the record.

Returns:

Is the trace clipped?

Return type:

bool

_get_results()#

Iterates through and runs _detect() on each trace in the stream to determine if the record is clipped or not.
